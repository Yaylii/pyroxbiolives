# Pyrox Bio Static Site

This version does not need npm, Next.js, or a build step.

## Deploy on Vercel
1. Create a new GitHub repo
2. Upload these files
3. Import the repo into Vercel
4. Deploy

## Files
- `index.html`
- `styles.css`

## What to edit
### Stock list
Edit the rows inside `index.html`

### Contact links
Replace both `href="#"` values near the bottom of `index.html`
with your real Telegram / DM links.
